# Schelling Segregation Model

## What this model does
This model simulates how agents with simple preferences for similar neighbors can lead to large-scale segregation patterns.

## Why I built this
To understand how local interactions can produce global emergent behavior.

## Mesa Features Used
- SingleGrid
- Neighbor queries
- Agent movement

## What I learned
- How local rules can lead to unexpected global outcomes
- Importance of neighbor interactions
- Emergent behavior in agent-based systems

## Challenges
- Understanding neighbor retrieval
- Handling agent movement

## Reflection
This model showed how simple constraints (preference for similar neighbors) can drastically affect system outcomes, reinforcing the importance of explicit constraint modeling.
