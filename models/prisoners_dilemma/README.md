# Prisoner's Dilemma Model

## What this model does
This model simulates agents randomly choosing between cooperation (C) and defection (D).

## Why I built this
To understand agent behavior and how simple rules affect system dynamics.

## Mesa Features Used
- MultiGrid
- RandomActivation scheduler
- Agent-based modeling

## What I learned
- How agents update state over time
- How grid-based systems work
- How local decisions impact global behavior

## Challenges
- Understanding Mesa structure
- Managing agent updates

## Reflection
This model helped me understand that constraints are embedded inside agent logic, which inspired my ActionSpace contribution.
